Install Instructions

  1. Install or Purchase Garry's Mod.
  2. Subscribe to Cinema on Steam Workshop. Also subscribe to the official map, cinema_theatron.
  3. Launch Garry's Mod and Click on "Sandbox" in the lower right hand side of GMod's menu.
  4. Select Cinema.
    Then select "Start New Game"
  5. Select Cinema on the left.
    Then select a supported map icon on the right.



  6. Select Singleplayer on the far right, if you want a solo game, or Multiplayer if you want to host a server!
  7. Enjoy the experience of Cinema!
